# 📜 Quotes API with GraphQL

This is a **GraphQL API** built with **Node.js, Apollo Server, and SQLite**. It allows users to **fetch random quotes, get quotes by author, and add new quotes**.

---

## 🚀 Features
- Fetch all quotes 📖
- Get a random quote 🎲
- Retrieve quotes by author ✍️
- Add a new quote ➕
